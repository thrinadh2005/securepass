=======
Make sure you have the required libraries installed. You can install them with: `pip install flask cryptography`

## How to Run
1. Double-click the `run_app.bat` file to start the application.
2. Alternatively, run `python app.py` from the command line.
3. The app will open in your default browser at http://127.0.0.1:5000/

## Creating a Desktop Shortcut
To create a desktop shortcut for easy access:
1. Right-click on `run_app.bat` and select "Create shortcut".
2. Move the shortcut to your desktop.
3. Double-click the shortcut to start the app anytime.
